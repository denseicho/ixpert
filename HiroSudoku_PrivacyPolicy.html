Privacy Policy

Last Updated: 2025/12/20

HiroSudoku (“the App”) respects and protects user privacy. This Privacy Policy explains how the App handles user data in compliance with applicable privacy laws and App Store requirements.

1. Data Collection

The App does not collect, store, share, or track any personal information. It does not collect:

  • Personal identity data (name, email, phone number, etc.)
  • Financial data
  • Location data
  • Contacts
  • User activity or analytics
  • Device identifiers (IDFA, advertising identifiers)
  • Usage data

2. Handling of Uploaded Images

All analysis of uploaded candlestick (K-line) images is performed locally on the user’s device.

  • No images are uploaded, stored, or transmitted externally.
  • No data is retained after analysis is completed.
  • No tracking is performed.

3. Data Sharing

The App does not share any user data with any third parties because it does not collect data in the first place.

4. Third-Party Services

The App does not use any third-party SDKs such as:

  • Analytics tools
  • Advertising networks
  • Cloud storage
  • Login services
  • Tracking or monitoring tools

5. Security

All processing is confined to the user’s device, ensuring maximum protection of user privacy.

6. Children’s Privacy

The App does not collect any data and does not contain inappropriate content. No information is knowingly collected from children.

7. Changes to this Policy

If future updates introduce new features involving data processing, this policy will be updated accordingly and notices will be provided within the App.
