プライバシーポリシー

最終更新日：2025年12月25日

HiroSudoku（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、適用されるプライバシー関連法令および App Store の要件に従って、ユーザーデータを適切に取り扱います。

1. 情報の収集について

本アプリ自体は、開発者がユーザーの個人情報を直接収集・保存・管理することはありません。

ただし、本アプリでは 広告配信のために第三者広告 SDK を使用する場合があります。 これらの第三者 SDK は、広告表示および最適化の目的で、デバイス識別子（広告識別子等）や 利用状況に関する情報を取得・処理することがあります。

  • 取得されるデータは、広告配信および測定の目的に限定されます
  • データの収集および処理は、各第三者 SDK のプライバシーポリシーに従って行われます
  • 開発者は、これらのデータを保存・管理・個別に特定することはありません

2. データの処理について

数独ゲームの進行や問題生成などの主要な処理は、ユーザーの端末内で行われます。 広告配信に関する処理のみ、第三者 SDK を通じて行われる場合があります。

3. データの共有について

本アプリは、開発者が収集したユーザーデータを第三者と共有することはありません。 ただし、広告配信に関しては、第三者広告 SDK が各社の規約に基づきデータを処理する場合があります。

4. 第三者サービスについて

本アプリでは、広告配信を目的として第三者サービス（広告 SDK）を利用することがあります。 解析ツール、クラウドストレージ、外部ログインサービス等は使用していません。

5. セキュリティについて

開発者は、ユーザーの個人情報を保持しない設計とし、アプリの安全性およびプライバシー保護に配慮しています。

6. お子様のプライバシーについて

本アプリは、13歳未満のお子様から意図的に個人情報を収集することはありません。

7. プライバシーポリシーの変更

広告 SDK の変更や法令の改定等により、本プライバシーポリシーを更新する場合があります。 変更があった場合は、本ページにて通知します。
